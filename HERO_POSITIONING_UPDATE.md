# Hero Slider Positioning Update

## Изменения

### Старая система
- Два выпадающих списка: "Позиция фона" и "Позиция объекта"
- Предустановленные значения (center, top, bottom, left, right и комбинации)
- Использовались CSS свойства `backgroundPosition` и `objectPosition`

### Новая система
- **Интерактивное визуальное позиционирование**
- Кликните или перетащите курсор по изображению для установки фокусной точки
- Красный крестик показывает текущую позицию
- Координаты сохраняются как X% и Y% (0-100)
- Поддержка touch-событий для мобильных устройств

## Технические детали

### Frontend (HeroSlideEditPage.tsx)
- Добавлен интерактивный контейнер с изображением
- Обработчики mouse и touch событий
- Визуальный индикатор позиции (красный крестик)
- Отображение координат в процентах

### Backend (HeroSlide.ts)
- Новые поля: `positionX` и `positionY` (Number, 0-100)
- Значения по умолчанию: 50% (центр)
- Старые поля сохранены для обратной совместимости

### Display (HeroSlider.tsx)
- Используется `backgroundPosition: ${positionX}% ${positionY}%`
- Плавная работа с новыми координатами
- Fallback к центру (50%, 50%) если данных нет

## Использование

1. Откройте админку → Hero Slides
2. Создайте или редактируйте слайд
3. Загрузите изображение
4. Кликните на изображение в нужном месте для установки фокусной точки
5. Перетаскивайте курсор для точной настройки
6. Красный крестик показывает, где будет центр изображения
7. Сохраните слайд

## Обратная совместимость

Существующие слайды со старыми полями `backgroundPosition` и `objectPosition` будут автоматически использовать значение по умолчанию (50%, 50% - центр). Рекомендуется переназначить позиции для точной настройки.

## Преимущества

✅ Визуальный контроль позиционирования
✅ Точное размещение фокусной точки
✅ Интуитивный интерфейс
✅ Поддержка touch-устройств
✅ Гибкость в настройке
